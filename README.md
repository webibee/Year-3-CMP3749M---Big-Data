# Nuclear Reactor Status Prediction & Flight Data Analysis
- Course: CMP3749M – Big Data
- Institution: University of Lincoln
- Assessment: Assessment 2 (90% of module mark)
- Grade Received: 65%
- Language: Python
- Frameworks: PySpark, Pandas, scikit-learn, Geopy


# Overview
This project comprises two distinct big data analytics tasks:

## Task 1: Nuclear Plant Status Prediction (70%)
Analysis of pressurised water reactor sensor data to predict whether a reactor is operating normally or abnormally. Three machine learning classifiers were implemented and evaluated:

- Decision Tree
- Support Vector Machine (SVM)
- Artificial Neural Network (Multilayer Perceptron)

## Task 2: Flight Data Analysis with MapReduce (30%)
Processing airline passenger and airport datasets using MapReduce paradigm in PySpark to:
- Count flights per airport
- Build flight manifests with passenger counts and arrival times
- Calculate nautical miles per flight and identify the passenger with the highest air miles
